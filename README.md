# react-country-icon
Component to render svg country flag

## Install

`npm i -D react-country-icon`

## Usage

```
import { CountryIcon } from 'react-country-icon';

const ExampleComponent = () => (
  <div>Hello <CountryIcon countryCode="CH"/> </div>
)
```
